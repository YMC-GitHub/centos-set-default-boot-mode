# how to use it with  developer?

```sh
# 编码脚本
#cat centos-set-default-boot-mode.sh
# 赋予权限
#chmod +x centos-set-default-boot-mode.sh
# 使用脚本
#2 作为解释器参数
bash ./centos-set-default-boot-mode.sh --help
#2 作为可执行程序
#3 切换到脚本所在目录
./centos-set-default-boot-mode.sh --help
#3 切换到其他目录执行
path/to/centos-set-default-boot-mode.sh --help
```
